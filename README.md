# Automated File Organizer with Python

## Overview
This project is a Python-based automation tool that organizes files.

## Features
- Automatic file classification by extension
- Batch file sorting into categorized folders
- Configurable rules for scalable preprocessing

## Tech Stack
- Python
- OS Module
- File System Automation

## Use Case in ML
In real ML systems, raw datasets must be cleaned and organized first.

## How to Run
```bash
python organizer.py
```
